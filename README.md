# DATA-PROFESIONALS-SURVEY
This project is a Power BI dashboard built to analyze insights from a survey of data professionals. The dashboard explores demographics, career paths, compensation, job satisfaction, and tools used across the data profession.  This project demonstrate skills in data cleaning, modeling, visualization, and insight generation using Power BI.

# Key Questions Answered

What is the average salary across different roles?

How does salary vary by gender?

Which data roles are most common?

What tools and technologies are most widely used by data professionals?

How satisfied are professionals with their current roles?

# Tools & Technologies

Power BI Desktop

Power Query (data cleaning & transformation)

DAX (measures and calculations)

Data Visualization & Dashboard Design
